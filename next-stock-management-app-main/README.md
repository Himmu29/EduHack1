# Next Stock Management App

This is a stock management app built with Next.js. It allows users to track and manage their inventory, monitor stock levels, and generate reports. The app provides a user-friendly interface for adding, updating, and deleting products, as well as viewing detailed information about each item. With Next.js, the app offers fast and efficient performance, server-side rendering, and easy deployment options. Whether you're a small business owner or a large-scale retailer, this stock management app is designed to streamline your inventory management process and improve overall efficiency.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
